* 정규표현식

        1. 정규 표현식?

        2. 객체 생성 방법
        - const regEx = new RexEx("식")
        - const regEx = / "식" /;

        3. 규칙
        ^ : 시작
        $ : 끝
        [] : 범위 [A-Z][a-z][가-힣][0-9]
        \d : 숫자
        \w : 영어 단어
        \s : 아무 공백 문자
        * : 0개 또는 여러 개
        + : 1개 이상
        ? : 0개 또는 1개
        . : 문자 1개
        {n,m} : n개 이상 m개 이하
