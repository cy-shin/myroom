        * 변수의 종류
        1. var
        - 변수
        - 중복해서 선언할 수 있음(중복 선언 시 덮어쓰기됨)
        - 함수 레벨
        
        2. 미작성
        - 항상 전역 변수

        3. let
        - 변수
        - 중복 선언 불가
        - 블록 레벨

        4. const
        - 상수
        - 중복 선언 및 값의 변경 불가
        - 블록 레벨

        주로 const, let을 사용하고 var는 사용을 피하는 것이 좋음
    
        * 자바스크립트의 자료형
        - 별도로 자료형을 지정하는게 아니라, 대입되는 값에 따라 자료형이 결정됨

        1. number

        2. String

        3. boolean

        4. object
        - 배열과 Map 형식의 JS 객체가 해당됨

        5. function
        - 함수

        6. undefient
        - 아직 값을 대입하지 않아 자료형이 결정되지 않았음

        7. null
        - 자료형이 아니라 참조하는 값이 없음을 의미함
